Описание. Данный REST-сервис спроектирван c использавнием среды разработки Intellij IDEA, Maven, SpringBoot, Java Servlet, применен плагин Lombok и библиотека Gson. Приложение на 80-м порту. Т.к мы используем SpringBoot, то приложение изначально разварачивается при помощи встроенного TomCat

Для запуска приложения необходимо запустить Intellij IDEA, открыть в нём проект, если не установлен плагин Lombok, дополнительно его установить, на машине так же должен быть установлен фреймворк сбоки Maven, необходимо выбрать конфигурацию Application и запустить её.

Для получения результатов (в формате JSON) в браузере набираем адрес http://localhost/hourse/rest/count, если мы хоти обратиться к RestController Spring или http://localhost/hourse/servlet/count, если к сервлету, c параметрами
width - ширина доски
height - высота доски
start - начальная расположения фигуры "Конь"
end - конечное расположение фигуры "Конь"
Ответ будет приходить в виде:

{"answer":3}

Где цифра будет обозначать минимальное количество ходов для достижения конечной точки
Если коненая точка не достижима, то будет возвращаться ответ  {"answer":-1}
Если введны некорректные данные, то будет возвращаться ответ  {"answer":-2} 

 Обозначение расположение фигуры аналогично таблице Excel, с ограничения по буквенной индексации английского алфавита. Проект содержит JUnit тесты.
